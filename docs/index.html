Installation

On this page I'm going to teach you how to install our client! For questions, join our discord server!

Downloading the Installer

First we need to download the installer. That's pretty easy, because all you have to do is click on download now. Then the installer will be installed!

Download Now

Running the Installer

The second step is to run the installer. To run the installer you need to double click on the installer app. Then a screen will appear. On that screen you have to follow all the steps that are asked!

And Play!

Now you have successfully installed ArtemisClient! Open our app ArtemisClient, Select the version. And play!

Troubleshooting

Are you having trouble installing/downloading the client? Feel free to join our discord server for more help. Or mail to info@artemisclient.net!
